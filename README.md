# mos-radio — Mo's Place Studio

## Included
- Separate development app
- RadioDJ-style interface and transparent logo
- Dashboard
- Personalities
- Voice Library with supplied Voicebox IDs
- Weekly schedule foundation
- Request settings for cooldown and per-user limits
- Safe placeholders for LiveDJ and News integration
- Local Windows EXE build script

## Run
Open `Studio` and double-click `Launch_Studio.bat`.

## Build EXE
Double-click `Tools\Build_Studio_EXE.bat`.

This build does not modify the live station.
