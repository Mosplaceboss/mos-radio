# Daily Operations Guide

Use the **Daily Operations** screen as your everyday home page.

## What you see

- **Station On Air** — whether a show is scheduled right now
- **Service status lights** — RadioDJ, Voicebox, LiveDJ, News, Requests, Website
- **Current host, format, and now playing**
- **Next scheduled event**
- **Today's advertising**
- **Alerts** — plain-English items that need attention

## Daily routine

1. Open **Daily Operations**.
2. Click **Refresh**.
3. Review alerts. Green means healthy, yellow means attention soon, red means help is needed.
4. Confirm the current host and format match expectations.
5. Use approved buttons only when needed:
   - **Open RadioDJ**
   - **Run News Now**
   - **Restart LiveDJ Watcher**
   - **Restart Request Watcher**

Every action asks for confirmation first.

## When something looks wrong

| Alert | What to do |
|-------|------------|
| RadioDJ stopped | Open RadioDJ or check the broadcast PC |
| Voicebox disconnected | Confirm Voicebox is running and the API address is correct |
| Watcher stopped | Restart the approved watcher from Daily Operations |
| Missing audio | Check the Audio Monitoring tab in Broadcasting |
| Stale news | Run News Now or check the News automation folder |

## Staff Mode

Staff Mode shows Daily Operations, Schedule, Personalities, Voice Library, Requests, Help, and limited Settings. Technical paths and development tools are hidden.
