# Mo's Place Studio — Quick Start Guide

Welcome to **Mo's Place Studio v2.0 Release Candidate**.

## What is Studio?

Mo's Place Studio is the control center for Mo's Place Radio. It helps you manage programming, personalities, automation, backups, and daily on-air operations without editing code or JSON files by hand.

## First launch

1. Run the installer or extract the portable ZIP.
2. Open **Mo's Place Studio** from the Desktop or Start Menu shortcut.
3. Complete the **First-Run Setup Wizard** when prompted.
4. Open **Daily Operations** to confirm RadioDJ, Voicebox, and automation health.

## Default platform folder

Studio uses this folder for shared station data:

`D:\MosPlaceRadioPlatform`

The installer creates required subfolders automatically. Your settings and station data are preserved during upgrades.

## Recommended first steps

1. Enter your station name and logo in Setup.
2. Confirm RadioDJ, music library, and Voicebox paths.
3. Test every connection until results are green or clearly explained in yellow.
4. Open **Help** for the daily checklist.

## Operating modes

| Mode | Who it is for |
|------|----------------|
| **Owner Mode** | Full production tools without advanced technical screens |
| **Staff Mode** | Simplified daily operations for on-air staff |
| **Advanced Mode** | Technical operators — paths, updates, and deep tools |

Change your mode in **Settings**.

## Need help?

Open **Help** inside Studio or read the other guides in the `docs` folder.
